# Obsidian Quickstart for Full Claude-Mem Corpus

## What to open
Open this folder as a vault:

`C:\Users\chris\Desktop\claude-mem-obsidian-full`

## How to see the visual graph
1. Open Obsidian.
2. Click **Open folder as vault**.
3. Choose `C:\Users\chris\Desktop\claude-mem-obsidian-full`.
4. Click **Graph view** (left sidebar network icon).

## How this vault is connected
- Every note links to:
  - its **Type** hub (`_hubs/types/...`)
  - its **Project** hub (`_hubs/projects/...`)
  - its **Session** hub (`_hubs/sessions/...`)
- This makes the full corpus graph connected and browsable.

## Best way to navigate
- Start with `Home.md`.
- Open `_hubs/Types Index.md` to slice by note type.
- Open `_hubs/Projects Index.md` to focus by project.
- Use backlinks panel on a hub note to see all linked notes.

## About Obsidian CLI
- The npm package command is `obsidian` (installed).
- It is for Obsidian QA/import workflows.
- For vault browsing and visual graph, the Obsidian desktop app is what you need.
