# WSL Setup Files

This folder contains shell scripts that were used for setting up the Amazon FBA Agent System on WSL (Windows Subsystem for Linux).

## Files Moved Here:
- `*.sh` - All shell scripts for Linux/WSL setup

## Current System Status:
The system now runs natively on Windows and these WSL setup files are no longer needed for normal operation.

## If You Want to Run on WSL:
1. Use the shell scripts in this folder
2. Install required dependencies using the setup scripts
3. Modify paths in configuration files to use Linux-style paths
4. Ensure Python and Node.js are installed in the WSL environment

## Note:
The main system is now optimized for Windows operation. WSL setup is provided for reference only.
